# Customer Churn Prediction & Customer Lifetime Value (LTV) Engine

## Project Overview

This project predicts customer churn and estimates Customer Lifetime Value (LTV) using Machine Learning techniques. The objective is to identify customers who are likely to leave a telecom company and help businesses improve customer retention strategies.
