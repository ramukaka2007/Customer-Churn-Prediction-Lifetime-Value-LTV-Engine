# Week 4 - Day 3

## Dashboard Planning

### Objective

Design an interactive dashboard for customer churn analysis and Customer Lifetime Value (LTV).

### Planned Dashboard Components

1. Customer Churn Overview
2. Churn by Contract Type
3. Monthly Charges Distribution
4. Customer Lifetime Value Distribution
5. Tenure Analysis
6. Senior Citizen vs Churn

### Filters

- Contract
- Payment Method
- Internet Service
- Senior Citizen
- Gender

### KPIs

- Total Customers
- Total Churn
- Churn Rate
- Average Monthly Charges
- Average LTV
